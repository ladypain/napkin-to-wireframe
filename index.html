Guía de Dibujo

  • 1. Dibuja libremente tus elementos
  • 2. Usa "Interpretar" para ver la versión refinada
  • 3. Repite el proceso para añadir más elementos

  • ⭕ Círculo = Radio button
  • ⭕• Círculo con punto = Radio button seleccionado
  • ☐ Cuadrado = Checkbox
  • ☑️ Cuadrado con tick = Checkbox seleccionado
  • ▭ Rectángulo = Input/Botón
  • Texto corto = Label
